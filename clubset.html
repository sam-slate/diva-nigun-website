7.20.23 Kleztronica NYC Set


This is a version of my "club" dance set that I play at Kleztronica shows, specifically modified for the 7.20.23 Kleztronica show in NYC. All the tracks in the set are Diva Nigun originals.



Directly before my set, the drag performer Chava GoodTime and I peformed a piece involving orca whales. The start of my set is a transition from the orca theme to dancing, and is sampling a D'Var Torah from an LA based rabbi recorded in 2019.
